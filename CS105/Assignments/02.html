Problem 1: Eliminating the Transfer Instruction

The transfer instruction, though useful, is not actually necessary. Show that any URM program can be transformed into one that is semantically equivalent and does not contain any transfer instructions.

Problem 2: Even or Odd

Write a URM program that, given a single input \(x\), has output \(1\) if \(x\) is even and \(0\) if \(x\) is odd.

Problem 3: Maximum

Write a URM program that, given inputs \(x\) and \(y\), outputs the maximum of \(x\) and \(y\).

Problem 4: Unused Registers

Argue that, given any program, there is always a register whose value that remains the same throughout its entire execution, even if the program does not terminate. In particular, given a program, write an informal procedure for determining an explicit index at which this holds.

Problem 5: Exponentiation (Challenge Problem)

Write a URM program that, given inputs \(x\) and \(y\), has outputs \(x^y\), the \(y\)th power of \(x\).

Problem 6: Some Basic Logic Questions

Turn the following statements into formulas in propositional logic, doing your best to determine which parts of the statements are propositions. You can use any letters you like for the individual propositions.

  • The garage door is open and there is no a car in the garage, but there is someone home.
  • The temperature of the water is in between 30 degrees and 70 degrees.

For each of the following formulas, provide an assignment that makes it true.

  • \(\neg ((A \land \neg B) \lor (B \land C)) \land C\)
  • \(\neg(\neg A \lor (\neg (\neg (B \lor A)) \lor C))\)

The second formula was supposed to read.

  • \(\neg(\neg A \lor (\neg (\neg (B \lor A) \lor C)))\)

Goes to show what one misplaced parenthesis can do. For your submission, you can either answer the question with the correct formula here or you can give a brief informal argument for why the second formula above does not have an assignment that makes it true.

Problem 7: Fewer Connectives

For functional completeness, we don't need both \(\lor\) and \(\land\).

Write a formula which is semantically equivalent to \(A \land B\) using only \(\neg\) and \(\lor\). Also write a formula which is semantically equivalent to \(A \lor B\) using only \(\neg\) and \(\land\).

We can actually get functional completeness with a single logical connective \(|\), the NAND connective. The formula \(\phi | \psi\) is semantically equivalent to \(\neg(\phi \land \psi)\). Write a formula which is semantically equivalent to \(\neg A\) using only \(|\). Also write a formula which is semantically equivalent to \(A \lor B\) using only \(|\).

Problem 8: Implication

Propositional logic usually includes a symbol \(\to\), where \(\phi \to \psi\) is interpreted as "if \(\phi\) then \(\psi\)" and is shorthand for \(\neg \phi \lor \psi\). Write a couple sentences about your thoughts on this definition. Do you feel that it adequately represents implication?
