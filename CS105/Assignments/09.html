Problem 1: argMax and argMin

Write an extension for Array including the following methods.

  • argMax<T>(_:, by: ) of generic type ((Element) -> T, (T, T) -> Bool) -> Element?, which returns the element in self that maximizes the first input function according to the less-than function given by the second input function. It should return nil if self is empty.
  • argMin<T>(_:, by: ) of generic type ((Element) -> T, (T, T) -> Bool) -> Element?, which returns the element in self that minimizes the first input function according to the less-than given by the second input function. It should return nil if self is empty. Try to write this function in one line.
  • argMax<T>(_:), a type constrained version of the above function in which T is comparable.
  • argMin<T>(_:), a type constrained version of the above function in which T is comparable.
