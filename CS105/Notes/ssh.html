Secure SHell and Secure CoPy

If you need to use UChicago's remove linux machine through the command line, you will need to use the ssh and scp commands. In the command line,

$ ssh <cnetID>@linux.cs.uchicago.edu

will open a shell on the remote machine for your account. From there you can type the same sorts of commands as you would on your local machine and they will be sent to the remote machine. You will be prompted for a password, use the one you use for Canvas. The command exit will bring you back to your local machine's command line.

If you need to get a file that you have locally on your machine to the remote machine

$ scp <fileName> <cnetID>@linux.cs.uchicago.edu:<pathOnRemoteMachine>

will do this. So

$ scp file.txt nmull@linux.cs.uchicago.edu:~

copies the file file.txt to the home directory on the remote machine. You will, again, be prompted for a password.

If you want to use these commands without using their password each time, follow the instructions on [this website] for setting up ssh keys.
